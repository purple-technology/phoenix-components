# Text Color

For some text components (<LinkTo kind="components-text--text">Text</LinkTo>, <LinkTo kind="components-paragraph--paragraph">Paragraph</LinkTo> and <LinkTo kind="components-heading--heading">Heading</LinkTo>) you can set different text colors (shades of gray) as a `color` prop:

- `white`
- `lightest`
- `light`
- `dark`
- `darkest`

<Story name="Text Color">
	<Text color="lightest">Lightest text</Text>
	<Text color="light">Light text</Text>
	<Text color="dark">Dark text</Text>
	<Text color="darkest">Darkest text</Text>
	<Spacer h="l" />
	<Paragraph color="lightest">Lightest paragraph</Paragraph>
	<Paragraph color="light">Light paragraph</Paragraph>
	<Paragraph color="dark">Dark paragraph</Paragraph>
	<Paragraph color="darkest">Darkest paragraph</Paragraph>
	<Spacer h="l" />
	<Heading color="lightest">Lightest heading</Heading>
	<Heading color="light">Light heading</Heading>
	<Heading color="dark">Dark heading</Heading>
	<Heading color="darkest">Darkest heading</Heading>
</Story>

You can also use for all of these components `colorTheme` prop which is described in more detail <LinkTo kind="docs-color-theme--color-theme">here</LinkTo>.
