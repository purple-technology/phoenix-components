# Spacing

Spacing values are standardized lenghts in pixels for setting paddings and margins in the form of `string`. These values are:

- `xxxs` - 4px
- `xxs` - 8px 
- `xs` - 12px
- `s` - 16px
- `m` - 20px
- `l` - 24px
- `xl` - 32px
- `xxl` - 40px
- `xxxl` - 52px

Visual representation of these spacings can be found in <a href="https://www.figma.com/file/DGocyaQPfb1oOy9eeGPUKQ/Phoenix-Design-System?node-id=2294%3A0" target="_blank">Phoenix Design System</a>.
